# 190. Reverse Bits

## Problem Description

Reverse bits of a given 32 bits signed integer.

## Examples

### Example 1:
```
Input: n = 43261596
Output: 964176192
Explanation:
Integer 43261596 in binary: 00000010100101000001111010011100
Integer 964176192 in binary: 00111001011110000010100101000000
```

### Example 2:
```
Input: n = 2147483644
Output: 1073741822
Explanation:
Integer 2147483644 in binary: 01111111111111111111111111111100
Integer 1073741822 in binary: 00111111111111111111111111111110
```

## Constraints

- `0 <= n <= 2^31 - 2`
- `n` is even.
